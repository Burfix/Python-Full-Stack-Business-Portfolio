# Supplier Pricing Intelligence Tool

Advanced supplier pricing analysis and invoice parsing system. Extract, analyze, and extract insights from supplier pricing data.

## Files
- `invoice_parser.py` - Initial parser implementation
- `invoice_parser_v2.py` - Enhanced parser with improved parsing logic
- `invoice_sample.csv` - Sample supplier pricing data for testing
- `INVOICE_PARSER_README.md` - Detailed documentation

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
python invoice_parser_v2.py
```

## Features
- Parse supplier invoice and pricing data from CSV files
- Support for multiple data formats
- Data validation and cleansing
- Price analysis and insights
